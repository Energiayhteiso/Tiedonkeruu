# Tiedonkeruu
Energiayhteisön käyttöön tiedonkeruun menetelmät Kamstrup omnia mittareilta sekä Fronius symo ja Kostal piko inverttereiltä. 

Tiedot lähetetään InfluxDB tietokantaan, sekä tallennetaan paikallisesti.
